# CSS Styling Task

## Description
This project demonstrates the use of HTML and CSS to apply different styles using element selectors, class selectors, id selectors, and nested selectors. It shows how styling can be applied to headings, paragraphs, lists, and links.

## Project Structure
project-folder/
index.html  
style.css  
README.md  

## How to Run
1. Download or save the project folder.
2. Make sure index.html and style.css are in the same folder.
3. Open index.html in any web browser.

## Features
- Uses class selector to style multiple headings
- Uses id selector for unique styling
- Applies styles to paragraph elements
- Demonstrates nested selectors for list items and links
- Shows different colors for ordered list items based on structure
- Applies background color to the whole page

## Technologies Used
HTML5  
CSS3  

## Output
When opened in a browser, the page will display:
- Headings with different styles using class and id
- Paragraphs with different colors
- Highlighted text inside headings and paragraphs
- Styled unordered and ordered lists using nested selectors
- A colored background for the page
